# 事件提醒系统 (Event Reminder System)

一个基于 Next.js 的事件管理和提醒系统，支持用户认证、事件管理和智能提醒功能。

## ✨ 主要功能

### 🔐 用户认证系统
- **用户登录**: 支持用户名/邮箱 + 密码登录
- **用户注册**: 新用户注册，默认创建普通用户账号
- **快速登录**: 提供演示账号快速登录按钮
- **用户会话**: 登录状态持久化，页面刷新后保持登录

### 👥 用户管理系统
- **用户列表**: 查看所有系统用户信息
- **添加用户**: 管理员可添加新用户账号
- **编辑用户**: 修改用户信息、角色和密码
- **删除用户**: 删除不需要的用户账号（不能删除当前登录用户）
- **权限控制**: 只有管理员可以访问用户管理功能

### 📅 事件管理
- **创建事件**: 添加新事件，包含标题、描述、截止日期和时间
- **状态管理**: 支持待处理、进行中、已完成三种状态
- **用户分配**: 为每个事件分配负责人
- **事件筛选**: 按状态筛选事件（全部/待处理/进行中/已完成）

### 🔔 智能提醒
- **自定义提醒**: 可设置提醒间隔（分钟）
- **过期提醒**: 自动检测并提示过期事件
- **实时通知**: 使用 sonner 库显示美观的通知
- **状态感知**: 已完成的事件自动停止提醒

### 💾 数据持久化
- **本地存储**: 使用 zustand persist 中间件
- **状态同步**: 页面刷新后数据不丢失
- **用户数据**: 用户认证状态和事件数据完整保存

## 🛠️ 技术栈

- **Frontend**: Next.js 15 (App Router)
- **UI Components**: shadcn/ui + Tailwind CSS
- **State Management**: Zustand + Persist
- **Notifications**: Sonner
- **Icons**: Lucide React
- **TypeScript**: 完整类型支持

### 🔑 登录系统

**普通用户账号**
- 邮箱: `zhangsan@demo.com` 或 用户名: `张三`
- 密码: `123456`
- 权限: 仅可访问事件管理

#### 快速登录
- 点击"快速登录：管理员"按钮自动填入管理员账号
- 点击"快速登录：普通用户"按钮自动填入普通用户账号

### 👥 用户管理 (仅管理员)

1. **查看用户列表**
   - 显示所有用户的基本信息
   - 用户角色标识（管理员显示皇冠图标）
   - 当前登录用户标记

2. **添加新用户**
   - 点击"添加用户"按钮
   - 填写用户名、邮箱、密码和角色
   - 系统会检查邮箱是否已被使用

3. **编辑用户信息**
   - 点击用户行的编辑按钮
   - 可修改用户名、邮箱、密码和角色
   - 密码字段留空则不修改密码

4. **删除用户**
   - 点击用户行的删除按钮
   - 确认删除后用户将被永久移除
   - 不能删除当前登录的用户

### 📋 事件管理

1. **添加事件**
   - 点击"添加事件"按钮
   - 填写事件标题、描述（可选）
   - 设置截止日期和时间
   - 选择负责人（从用户列表选择）
   - 设置事件状态
   - 配置提醒设置

2. **管理事件**
   - 使用筛选按钮查看不同状态的事件
   - 点击"开始处理"将事件状态改为进行中
   - 点击"标记完成"将事件状态改为已完成
   - 已完成的事件可以"重新打开"
   - 点击删除按钮移除事件

3. **事件提醒**
   - 距离截止时间到达设定的提醒间隔时会弹出通知
   - 事件过期后会显示过期提醒
   - 已完成的事件不再发送提醒

## 🚨 注意事项

1. **演示项目**: 这是一个演示项目，密码等敏感信息存储在浏览器本地存储中
2. **数据安全**: 实际生产环境需要实现后端认证和数据库存储
3. **浏览器兼容**: 建议使用现代浏览器以获得最佳体验
4. **数据持久**: 清除浏览器数据会丢失所有本地存储的信息

## 🔧 开发说明

### 状态管理结构
```typescript
// 用户状态
interface User {
  id: string
  name: string
  email: string
  password: string
  role: 'admin' | 'user'
  createdAt: string
}

// 事件状态
interface Event {
  id: string
  title: string
  description: string
  createdAt: string
  deadline: string
  time: string
  status: 'pending' | 'in-progress' | 'completed'
  assignedUserId: string
  reminderEnabled: boolean
  reminderInterval: number
}
```

### 权限验证
```typescript
// 检查是否为管理员
const isAdmin = currentUser?.role === 'admin'

// 条件渲染
{isAdmin && <UserManagement />}
```
