# 事件提醒系统功能测试

## 🧪 测试步骤

### 1. 用户认证测试

#### 登录功能测试
1. 访问 http://localhost:3000
2. 应该看到登录界面
3. 测试快速登录：
   - 点击"快速登录：管理员"，表单应自动填入 admin@demo.com 和 123456
   - 点击"快速登录：普通用户"，表单应自动填入 zhangsan@demo.com 和 123456
4. 手动输入测试：
   - 输入用户名 "管理员" 和密码 "123456"，应该能成功登录
   - 输入邮箱 "admin@demo.com" 和密码 "123456"，应该能成功登录
5. 错误处理测试：
   - 输入错误密码，应该显示错误提示
   - 输入不存在的用户，应该显示错误提示

#### 注册功能测试
1. 在登录页面点击"没有账号？点击注册"
2. 填写注册表单：
   - 用户名：test_user
   - 邮箱：test@example.com
   - 密码：123456
   - 确认密码：123456
3. 点击注册，应该成功并跳转回登录页面
4. 用新注册的账号登录，应该成功
5. 验证新用户默认为普通用户权限（不能访问用户管理）

### 2. 事件管理测试

#### 添加事件测试
1. 以管理员身份登录
2. 在事件管理页面点击"添加事件"
3. 填写事件信息：
   - 标题：测试事件
   - 描述：这是一个测试事件
   - 截止日期：选择明天
   - 截止时间：选择当前时间+1小时
   - 负责人：选择任意用户
   - 提醒：开启，间隔30分钟
4. 点击添加，应该成功创建事件并显示在列表中

#### 事件状态管理测试
1. 创建一个新事件（状态为"待处理"）
2. 点击"开始处理"按钮，状态应该变为"进行中"
3. 点击"标记完成"按钮，状态应该变为"已完成"
4. 点击"重新打开"按钮，状态应该变回"待处理"

#### 事件筛选测试
1. 创建多个不同状态的事件
2. 点击筛选按钮：
   - "全部"应显示所有事件
   - "待处理"只显示待处理的事件
   - "进行中"只显示进行中的事件
   - "已完成"只显示已完成的事件
3. 筛选按钮应显示正确的数量统计

### 3. 用户管理测试（仅管理员）

#### 用户列表查看
1. 以管理员身份登录
2. 切换到"用户管理"标签页
3. 应该看到所有用户的列表，包括：
   - 用户名、邮箱、角色、创建时间
   - 当前登录用户有"当前用户"标记
   - 管理员有皇冠图标

#### 添加用户测试
1. 点击"添加用户"按钮
2. 填写用户信息：
   - 用户名：新用户
   - 邮箱：newuser@example.com
   - 密码：123456
   - 角色：普通用户
3. 点击添加，应该成功创建用户
4. 尝试添加重复邮箱，应该显示错误提示

#### 编辑用户测试
1. 点击任意用户的编辑按钮
2. 修改用户信息
3. 点击保存，应该成功更新
4. 验证信息是否正确更新

#### 删除用户测试
1. 点击非当前用户的删除按钮
2. 确认删除，用户应该从列表中移除
3. 尝试删除当前登录用户，应该显示错误提示

### 4. 权限控制测试

#### 普通用户权限测试
1. 以普通用户身份登录（如张三）
2. 应该只能看到"事件管理"标签
3. "用户管理"标签应该有"管理员"标记
4. 点击用户管理标签，应该显示"权限不足"提示

### 5. 提醒功能测试

#### 提醒设置测试
1. 创建一个截止时间在30分钟内的事件
2. 确保提醒功能开启，间隔设置为30分钟
3. 等待提醒时间到达，应该弹出提醒通知

#### 过期提醒测试
1. 创建一个截止时间已过的事件
2. 应该显示"已过期"标记
3. 应该弹出过期提醒通知

### 6. 数据持久化测试

#### 刷新页面测试
1. 登录系统并创建一些事件
2. 刷新页面
3. 应该保持登录状态
4. 所有数据应该保持不变

#### 重新打开浏览器测试
1. 完全关闭浏览器
2. 重新打开并访问系统
3. 应该保持登录状态和所有数据

### 7. UI响应式测试

#### 移动端适配测试
1. 打开浏览器开发者工具
2. 切换到移动设备视图
3. 所有界面应该正确适配移动端

#### 不同分辨率测试
1. 调整浏览器窗口大小
2. 界面应该响应式调整

## 🐛 常见问题排查

### 登录问题
- 检查是否使用了正确的演示账号
- 确认密码输入正确（区分大小写）
- 检查浏览器控制台是否有错误

### 数据丢失
- 检查浏览器是否禁用了localStorage
- 确认没有手动清除浏览器数据
- 检查是否在无痕模式下使用

### 提醒不工作
- 确认浏览器允许通知
- 检查事件的提醒设置是否正确
- 确认系统时间设置正确

### 界面显示问题
- 刷新页面重试
- 检查网络连接
- 确认浏览器版本支持
